# Мост для публикации постов в Телеграм-чат из сообщества в ВК
### _Как это работает?_
- ВК бот получает команду "отправить"
- Ожидается следующее сообщение, как текст поста
- При получении идёт попытка отправки сообщения в чат Телеграм
- При успешной отправке бот в ВК отвечает "Отправлено!", а в консоле появляется debug-сообщение с полной датой и текстом сообщения

> [!WARNING]
> ### Перед запуском: 
> - включите Long Poll API в настройках сообщества
> - в том же LP API во вкладке "Типы событий" включите все галочки для сообщений
> - проверьте корректность данных в файле config.py

Запуск осуществляется командой: `python main.py`
